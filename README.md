# Google Keep

Simplified copy Google Keep

Используется:
* TypeScript
* React.js
* Redux
* Node.js

Использовать:
* npm i
* && npm start чтобы запустить приложение

Реализовано:
* верстка
* создание/просмотр/редактирование/удаление заметки 
* удаление всех заметок
* поиска по заметкам
